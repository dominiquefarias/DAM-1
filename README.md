# 🚀 Ciclo DAM (Desarrollo de Aplicaciones Multiplataforma)

Este repositorio contiene la colección de ejercicios, prácticas y proyectos desarrollados durante el Ciclo Formativo de Grado Superior en **Desarrollo de Aplicaciones Multiplataforma (DAM)**. El contenido abarca desde los fundamentos de la programación hasta el despliegue de aplicaciones con bases de datos y entornos de servidor.

## 📂 Estructura del Repositorio

El proyecto se organiza siguiendo la progresión académica del ciclo:

### 📗 1º Trimestre: Fundamentos y Lógica
Enfoque en la adquisición de bases sólidas en programación y gestión de datos.
* **Programación:** Primeros pasos con **Python** (estructuras de control, funciones, manejo de archivos con `pickle`).
* **Bases de Datos:** Diseño y creación de esquemas en **MySQL** y **SQLite**. Gestión de usuarios, privilegios y operaciones CRUD fundamentales.
* **Desarrollo Web:** Introducción a **HTML5** y microservicios básicos con **Flask**.
* **Sistemas:** Configuración de entornos y scripts de automatización.

### 📘 2º y 3º Trimestre: Desarrollo Avanzado e Integración
Profundización en tecnologías backend y frontend, integrando sistemas complejos.
* **Backend:** Desarrollo con **PHP 8.2+**, enfocándose en la seguridad (sentencias preparadas) y la gestión de sesiones.
* **Frontend Interactivo:** Interfaces modernas con **JavaScript Vanilla**, manipulación dinámica del DOM y eventos avanzados.
* **Bases de Datos Avanzadas:** Consultas complejas (JOINs), subconsultas, agrupamientos y experimentación con bases de datos NoSQL y vectoriales (**MongoDB**, **ChromaDB**).
* **IA & Innovación:** Integración de modelos de lenguaje locales (**Ollama**) para la generación de código y asistentes inteligentes.

## 🚀 Proyectos Destacados

* **Task Master:** Herramienta integral de gestión de tareas para estudiantes con interfaz minimalista.
* **Gestor de Clientes:** Aplicación en Python con persistencia de datos.
* **SGE & E-commerce:** Desarrollo de módulos para sistemas de gestión empresarial y tiendas online funcionales.

## ⚙️ Requisitos de Ejecución

Para ejecutar la mayoría de las prácticas web y de backend, se requiere:
1. Un entorno **Linux Ubuntu** (o similar).
2. Servidor local **Apache** con soporte para PHP 8.2.
3. Intérprete de **Python 3.x**.
4. Servidor **MySQL** activo.
